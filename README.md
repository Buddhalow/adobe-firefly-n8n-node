# n8n-nodes-adobe-firefly

This is a custom node for the [n8n](https://n8n.io/) workflow automation tool. It allows you to generate images using the Adobe Firefly API.

## Installation

You can install this custom node in n8n by following these steps:

1. Open your n8n instance.
2. Go to the "Workflow" menu and select "Settings".
3. In the "Custom Nodes" tab, click on "Add Custom Node".
4. Enter the GitHub URL for this repository: `https://github.com/john-s-doe/n8n-nodes-adobe-firefly.git`
5. Click "Save" to install the custom node.

## Usage

This custom node provides a single node that can be used to generate images using the Adobe Firefly API.

1. Add the "Adobe Firefly" node to your n8n workflow.
2. Configure the node with your Adobe Firefly API credentials.
3. Use the node to generate images.

Please note that you will need to have an Adobe Firefly API account and have an API key to use this node.
